# Zustand State Management

Sanju Career OS uses Zustand stores.

## Active Stores
1. **useCareerStore** (`frontend/src/app/store/useCareerStore.ts`): Stores user profile, daily logs, LeetCode checklist progress, projects, job applications, achievements, and CS Core checkpoints.
2. **useUIStore** (`frontend/src/app/store/useUIStore.ts`): UI states like sidebar collapsed states, active page navigation tabs, and achievements overlays.
3. **useAIStore** (`frontend/src/app/store/useAIStore.ts`): AI chat logs.
