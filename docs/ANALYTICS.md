# Career Analytics Calculations

Calculates career readiness progress metrics.

## Scoring Formulas
- **Placement score**: 30% DSA + 15% SkillRack + 15% Aptitude + 10% SQL + 15% CS Core + 10% Projects + 3% Resume + 2% Consistency.
- **Germany readiness score**: German study progress is tracked separately from placement readiness.
- **Consistency index**: Based on streak and logs counts completed.
