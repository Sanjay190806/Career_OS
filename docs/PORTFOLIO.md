# Portfolio Placement Showcase

Detailed project overview for engineering resumes and campus placement showcases.

## Project Summary
Sanju Career OS is an AI-powered Career Operating System designed to help engineering students track placement readiness. It features daily trackers, DSA roadmaps, projects, resume analysis, and a chatbot.

## Resume Bullet Points
- Designed and built a full-stack Career Operating System in React and Express, enabling students to log daily placement trackers.
- Implemented real-time streaming AI mentor completions using Server-Sent Events (SSE) securely mapped with Groq Llama-3 API.
- Integrated PostgreSQL synchronization backups layer using Prisma ORM to persist and restore progress.
